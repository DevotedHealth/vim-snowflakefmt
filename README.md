# vim-snowflakefmt

Run `snowflakefmt` on sql code

## Installation

```vim
Plugin 'DevotedHealth/vim-snowflakefmt'
```

## Usage

Run `:w` to format automatically. If you set `g:sqlfmt_auto` to 0, this behavior will be disabled.

It can also be executed directly as follows:

```vim
:SQLFmt [files...]
```
